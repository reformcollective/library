# Infinite Side Scroll

Using the native horizontal scrolling, allows you scroll, or use the buttons, to progress through the content FOREVER.

Pass in Gradient as a styled div with a gradient background and desired size.  InfiniteSideScroll will position and place it correctly.

Pass in Button as a styled button or UniversalLink as if it were the increment button.  InfiniteSideScroll will create the deprecate button with scaleX(-1)

## Custom Styling

InfiniteSideScroll has a className prop to allow you to add custom styling if needed.  You can style the inner elements like so:

```js
  import { Inner, Track, GradientRight, GradientLeft } from 'library/InfiniteSideScroll'

  const StyledSideScroll = styled(InfiniteSideScroll)`
    // custom styling for Wrapper

    ${Inner} {
      // custom styling for Inner
    }

    ${Track} {
      // custom styling for Track
    }

    ${GradientRight} {
      // custom styling for GradientRight
      // should only use to override default styles:
      /*
        position: fixed;
        z-index: 2;
        top: 0;
        right: 0;
        transform: scaleX(-1);
      /*
    }

    ${GradientLeft} {
      // custom styling for GradientLeft
      // should only use to override default styles:
      /*
        position: fixed;
        z-index: 2;
        left: 0;
        top: 0;
      /*
    }
  `
```
Button styles are only applied on the styled component that you are passing in as a prop.


## Example Usage

```js
  <InfiniteSideScroll 
    trackGap={50}
    Gradient={Gradient}
    Button={StyledButton}
  >
    <Card $color="purple"/>,
    <Card $color="indigo"/>,
    <Card $color="blue"/>,
    <Card $color="green"/>,
    <Card $color="lightGreen"/>,
    <Card $color="yellow"/>,
    <Card $color="gold"/>,
    <Card $color="orange"/>,
    <Card $color="orangered"/>,
    <Card $color="red"/>,
  </InfiniteSideScroll>


  const Card = styled.div<{ $color: string }>`
    width: 300px;
    height: 300px;
    background: ${({ $color }) => $color};
  `

  const Gradient = styled.div`
    width: 100px;
    height: 300px;
    background: linear-gradient(
      to right,
      white 20%,
      transparent 100%
    );
  `

  const StyledButton = styled.button`
    width: 50px;
    height: 50px;
    font-size: 20px;
    cursor: pointer;
  `
```
